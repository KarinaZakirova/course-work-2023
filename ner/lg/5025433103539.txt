Белым по белому — снегом по вечности —
Пишется чья-то судьба.
Синим по синему — волей по верности,
А в отраженьях — душа.
Серым по серому — пеплом по времени —
Память навеки одна.
Красным по красному — кровью по ревности
<Горечь> не выжечь дотла.
Жёлтым по жёлтому — светом по гордости —
Сила природой дана.
Зелень по зелени — злостью по робости —
Только дорога длинна.
Тёмным по тёмному – необретённая
<Истина> снова ушла.
Разным по разному – с волей и разумом
Переплетутся слова.
